Q01Code Trace — df mean
import pandas as pd
df = pd.DataFrame({"item":["pen","book"],"price":[1.5,12.0]})
print(df["price"].mean())
  • a13.5
  • b1.5
  • c6.75
  • d12.0
Explanation

(1.5 + 12.0) / 2 = 13.5 / 2 = 6.75. Verbatim Q1.

Q02MC — head()
What does df.head() return?
  • aLast 5 rows
  • bColumn names only
  • cFirst 5 rows
  • dNumber of rows
Explanation

df.head() returns the first 5 rows. df.tail() = last 5. df.columns = column names. Verbatim Q2.

Q03MC — read_csv
Which method loads a CSV file into a DataFrame?
  • apd.read_table()
  • bpd.load_csv()
  • cpd.read_csv()
  • dpd.open()
Explanation

pd.read_csv() is the correct method. pd.load_csv() and pd.open() do not exist. Verbatim Q3.

Q04Code Trace — boolean filter
df = pd.DataFrame({"name":["Alice","Bob","Carol"],"score":[85,72,91]})
result = df[df["score"] > 80]
print(result["name"].values)
  • a["Alice","Bob","Carol"]
  • b["Bob"]
  • c["Alice","Carol"]
  • dError
Explanation

df["score"] > 80 → [True,False,True]. Alice(85✓), Bob(72✗), Carol(91✓). Output: ["Alice","Carol"]. Verbatim Q4.

Q05MC — describe()
What does df.describe() return for a numeric column?
  • aA bar chart
  • bColumn data types
  • cSummary stats: count, mean, std, min, max
  • dFirst 5 rows
Explanation

describe() → count, mean, std, min, 25%, 50%, 75%, max. Verbatim Q5.

Q06MC — fillna
Which method fills missing values with a specified value?
  • adf.dropna()
  • bdf.replace()
  • cdf.fillna()
  • ddf.clean()
Explanation

df.fillna() fills NaN with a value. df.dropna() removes rows with NaN. df.clean() does not exist. Verbatim Q6.

Q07Code Trace — groupby mean
df = pd.DataFrame({"dept":["HR","IT","HR","IT"],"salary":[60000,90000,65000,95000]})
print(df.groupby("dept")["salary"].mean())
  • aHR:60000, IT:90000
  • b77500
  • cHR:62500, IT:92500
  • dError
Explanation

HR: (60000+65000)/2 = 62500. IT: (90000+95000)/2 = 92500. Verbatim Q7.

Q08MC — isnull
Which method shows the number of missing values per column?
  • adf.describe()
  • bdf.isnull().sum()
  • cdf.head()
  • ddf.info()
Explanation

df.isnull().sum() returns a Series with null counts per column. Verbatim Q8.

Q09MC — sort_values
What is the key difference between df.sort_values("price") and list.sort()?
  • asort_values() sorts alphabetically
  • bThey are identical
  • csort_values() returns a new df; list.sort() modifies in place
  • dsort_values() only works on numbers
Explanation

sort_values() returns a new DataFrame — must assign to save. list.sort() modifies the list in place and returns None.

Q10MC — dropna
What does df = df.dropna() do?
  • aFills NaN with zero
  • bModifies df in place without assignment
  • cReturns a new df with NaN rows removed, assigned back
  • dRemoves columns with NaN
Explanation

dropna() returns a new DataFrame. Removes ROWS (not columns) with NaN. The assignment df = df.dropna() saves the result.
