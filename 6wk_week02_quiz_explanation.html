Q01Code Trace — if/elif/else
score = 85
if score >= 90:
print("A")
elif score >= 80:
print("B")
else:
print("C")
  • aA
  • bB
  • cC
  • dError
Explanation

85 < 90 so first condition fails. 85 >= 80 is True so elif runs and prints "B". Verbatim Q1.

Q02Code Trace — for loop sum
total = 0
for i in range(1, 6):
total += i
print(total)
  • a6
  • b10
  • c15
  • d21
Explanation

range(1,6) = [1,2,3,4,5]. Sum = 1+2+3+4+5 = 15. Verbatim Q2.

Q03MC — elif keyword
Which keyword introduces an alternative condition after if?
  • aelse
  • bthen
  • celif
  • delseif
Explanation

elif (else-if) introduces an additional condition. Python does not use "then" or "elseif". Verbatim Q3.

Q04Code Trace — function with return
def compute_tax(price, rate):
return price * rate
result = compute_tax(100, 0.07)
print(result)
  • a7
  • b107
  • c0.07
  • d7.0
Explanation

100 * 0.07 = 7.0 (float because rate is float). The function returns 7.0.

Q05MC — break usage
What does break do inside a loop?
  • aSkips to the next iteration
  • bExits the loop immediately
  • cRestarts the loop from the beginning
  • dPauses execution
Explanation

break exits the loop immediately. continue skips to the next iteration. break is used to exit early when a condition is met.

Q06Code Trace — def greet
def greet(name):
message = f"Hello, {name}!"
return message
print(greet("Alice"))
  • aHello, name!
  • bmessage
  • cHello, Alice!
  • dError
Explanation

greet("Alice") calls the function with name="Alice". f"Hello, {name}!" → "Hello, Alice!". The function returns it. print() displays it.

Q07MC — while loop
When does a while loop stop executing?
  • aAfter exactly 10 iterations
  • bWhen its condition becomes False
  • cWhen break is called
  • dBoth b and c
Explanation

A while loop stops when its condition is False OR when break is explicitly called. Both are valid exit mechanisms.

Q08Code Trace — conditional business
discount = 0
total = 120
if total > 100:
discount = 10
print(discount)
  • a0
  • b10
  • c120
  • dError
Explanation

120 > 100 is True, so discount = 10 executes. Output: 10. Verbatim (Week 2 Q3).

Q09MC — return vs print
What is the difference between print() and return inside a function?
  • aThey are identical
  • bprint() shows output; return sends a value back to the caller
  • creturn shows output; print() sends a value back
  • dreturn only works inside loops
Explanation

print() displays to the screen but the value is not usable by the calling code. return sends the value back so it can be stored, used in calculations, or passed to other functions.

Q10Code Trace — range
for i in range(3):
print(i)
  • a1 2 3
  • b0 1 2 3
  • c0 1 2
  • dError
Explanation

range(3) = [0, 1, 2]. range(n) starts at 0 and goes up to (not including) n. Prints: 0, 1, 2 on separate lines.
