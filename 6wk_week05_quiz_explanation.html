Q01MC — groupby sum
What does df.groupby("region")["sales"].sum() return?
  • aRow count per region
  • bTotal sales for each region
  • cAverage sales per region
  • dA list of region names
Explanation

sum() = total. count() = row count. mean() = average. Verbatim Q1.

Q02MC — plt.bar
Which matplotlib function creates a bar chart?
  • aplt.line()
  • bplt.bar()
  • cplt.scatter()
  • dplt.hist()
Explanation

plt.bar() = bar chart. plt.line() DOES NOT EXIST — use plt.plot() for lines. Verbatim Q2.

Q03MC — line chart use case
When is a line chart the best choice?
  • aComparing values across categories
  • bShowing the distribution of a variable
  • cShowing a trend over time
  • dShowing two unrelated variables
Explanation

Line chart = trend over time. Comparing categories = bar chart. Distribution = histogram. Verbatim Q3.

Q08MC — mean vs median
If df["revenue"].mean() = 1200 and df["revenue"].median() = 650, what does this suggest?
  • aData is symmetric
  • bData is left-skewed
  • cData is right-skewed — large transactions pull the mean up
  • dThere is an error
Explanation

Mean > median = right-skewed. A few very large transactions pull the mean upward. Median ($650) is more representative of a typical transaction.

Q09MC — savefig
Why use index=False in df.to_csv("file.csv", index=False)?
  • aMakes the file smaller
  • bPrevents writing the row index (0,1,2...) as an extra column
  • cRequired — to_csv fails without it
  • dSkips rows with NaN
Explanation

Without index=False, pandas writes the row numbers as "Unnamed: 0" in the output file. Always use index=False.

Q10MC — str.title cleaning
Why apply df["region"].str.title() before groupby("region")?
  • agroupby only works on title-case columns
  • bIt makes output look nicer
  • c"north","North","NORTH" are treated as three separate groups without normalization
  • dIt converts strings to numbers
Explanation

pandas groupby is case-sensitive. Normalizing with .str.title() ensures all regions merge into one group per region. This is one of the most common real-world data bugs.
